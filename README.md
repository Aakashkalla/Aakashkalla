## 🚀 Projects

### [Smart Complaint Management System](https://github.com/Aakashkalla/smart-complaint-system) · [Live](https://smart-complaint-system-indol.vercel.app/)
> Civic complaint platform where citizens report issues with image evidence and real-time tracking.
- Integrated **Mistral 7B via OpenRouter API** to auto-classify complaints into municipal departments — zero manual routing
- Built multi-role admin dashboard with status management, category filters, and resolution tracking
- Automated email pipeline on submission and status updates using **Resend**
- Stack: `Next.js` `TypeScript` `Supabase` `PostgreSQL` `Prisma` `Tailwind CSS`

---

### [Room Finder](https://github.com/Aakashkalla/room-finder) · [Live](https://room-finder-one.vercel.app/landing)
> Two-sided rental marketplace — tenants find rooms, landlords list and manage properties.
- Role-aware auth with **Supabase Auth** — users self-select as Tenant or Owner at signup with server-enforced route protection
- Handled image uploads, secure file access, and multi-listing management via **Supabase Storage**
- Stack: `Next.js` `TypeScript` `Supabase` `PostgreSQL` `Tailwind CSS`

---

### [n0track — Real Time Chat](https://github.com/Aakashkalla/chat-frontend) · [Live](https://n0tracker.vercel.app/)
> Anonymous real-time chat with room creation, live presence, and message broadcasting.
- Engineered WebSocket system with live join/leave events, timestamped messages, and user presence tracking
- Split deployment: frontend on **Vercel**, backend on **Render** with health checks and environment config
- Stack: `React` `TypeScript` `Node.js` `Express` `Socket.IO` `Tailwind CSS`
